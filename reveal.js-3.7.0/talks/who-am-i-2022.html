What is a memory?

First, neurons

Parts of a neurons

80-100 billion neurons, over 60 trillion connections

Memory == neuron activation patterns

What is deja vu?

A storage "collision"

Memory is an activation pattern, but experiencing events is also an activation pattern

How do experiences translate to memory?

Repetition, novelty, and "importance"

Consolidation

Sleep "replays" recent activation patterns

Adrenaline (epinephrine) strengthens recent activation patterns

Is memory static?

Recall activates specific neuron patterns

Activation patterns depend on strength of connections between neurons

Connections strengths constantly change (neural plasticity)

How we recall events changes every time we recall them
